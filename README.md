# 卡塞尔学院守夜人社区 · 邮件情报后台

> 《龙族》同人作品 —— 复古欧式贵族秘党风格的邮件情报后台 UI。

在线地址：https://lxl1234-lxl.github.io/Collegium-Cassellianum/

## 这是什么

这是卡塞尔学院守夜人社区的邮件情报后台，模拟《龙族》世界观中专员接收、处理学院邮件与任务简报的日常。整体采用羊皮纸、橄榄墨绿与鎏金的复古学院配色，并可在“学院日常”与“高阶作战”两种氛围之间一键切换。

## 主要功能

- **双端响应式**：PC 大屏三栏桌面布局 + 手机竖屏底部 Tab + 侧边抽屉菜单。
- **学院 / 作战 双主题**：左下角切换按钮，整体色系、水印、氛围随主题变化。
- **邮件情报**：查看来自执行部、装备部、信息部的邮件与任务简报。
- **联系人列表**：左侧/抽屉展示联系人头像、名称与留言预览。
- **重要邮件**：点击星标将邮件归入“重要邮件”，随时在重要邮件列表查看。
- **回复留言**：对任意邮件撰写并保存回复内容。
- **删除提示**：点击删除时弹出学院指令弹窗，提示“一切情报痕迹不得清除，仅可存档”。
- **个人编号**：界面中显示专员编号 `SSS0024076`。

## 技术说明

- React 18 + TypeScript + Vite + Tailwind CSS
- 状态管理：Zustand
- 路由：HashRouter（便于单文件与静态部署兼容）
- 构建产物为单个 `index.html`，资源内联，方便直接部署到 GitHub Pages 或在手机浏览器打开。
- 本项目为纯前端演示，不含后端服务、API Key 或任何个人隐私数据。
